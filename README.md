# Loadbalancer with nodejs containers

> using HAProxy

This repo is the implementation of Hussein Nasser's YouTube [tutorial](https://youtu.be/9sAg7RooEDc).

## Usage

1. clone the repo
1. use `docker-compose up` to spin up the containers
